# ☕ Cafetería Web - Proyecto Frontend

Bienvenido a este proyecto de cafetería 👋
Esta página web representa el diseño visual de una cafetería moderna, enfocada en la experiencia del usuario y la presentación atractiva de productos.

---

## 🧾 Sobre el proyecto

Este proyecto consiste en una página web que simula una cafetería digital.
Está desarrollado únicamente a nivel **frontend**, por lo que no incluye funcionalidades reales como pedidos, pagos o conexión a base de datos.

El enfoque principal es el diseño, la estructura y la estética del sitio.

---

## 🎨 Características

* Diseño moderno y minimalista
* Sección de presentación de la cafetería
* Catálogo visual de productos (bebidas y alimentos)
* Uso de imágenes atractivas
* Interfaz responsive (adaptable a dispositivos móviles y escritorio)
* Animaciones y efectos visuales

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (para efectos visuales básicos)

---

## 🎯 Objetivo

El propósito de este proyecto es:

* Practicar diseño web enfocado en negocios reales
* Mejorar habilidades en maquetación frontend
* Crear una interfaz atractiva y funcional
* Desarrollar un proyecto visual para portafolio

---

## 📂 Estructura del proyecto

* `index.html` → Página principal que es el login
* `login.html` → Con su respectivo estilo
* `cafeteria.html` → Contenido principal
* `style.css` → Estilos del sitio
* `script.js` → Interacciones visuales
* `img/` → Recursos gráficos
